## 📚 Короткі теоретичні відомості
Курс **Exploring Networking with Cisco Packet Tracer** призначений для ознайомлення з основами мережевих технологій через симуляцію в Cisco Packet Tracer. У модулі 1 ви дізнаєтеся, як налаштувати офісну мережу, використовуючи структуровану кабельну систему та бездротові технології (Wi-Fi, Bluetooth, стільникові мережі). У модулі 2 ви навчитеся керувати та моніторити мережу за допомогою симуляційного режиму Packet Tracer та мережевого контролера. Курс включає практичні лабораторні роботи для створення, налаштування та аналізу мереж. Симуляція дозволяє тестувати підключення, трафік та конфігурацію без фізичного обладнання.

### 2. Словник термінів
Нижче наведено таблицю з 10 новими термінами, вивченими в курсі.

| Термін (англ.)            | Термін (укр.)             | Визначення (пояснення)                                                                 |
|---------------------------|---------------------------|---------------------------------------------------------------------------------------|
| Structured Cabling        | Структурована кабельна система | Організований підхід до створення кабельної інфраструктури мережі, що включає використання патч-панелей, настінних роз’ємів і кольорових кабелів для ефективного підключення пристроїв (1.1.2). |
| Packet Tracer             | Пакетний трасер           | Програма від Cisco для симуляції мереж, що дозволяє створювати, налаштовувати та тестувати мережеві топології без фізичного обладнання (1.0.2). |
| CLI (Command Line Interface) | Інтерфейс командного рядка | Інтерфейс для конфігурації мережевих пристроїв Cisco через текстові команди в операційній системі IOS (1.1.7). |
| Wireless LAN (WLAN)       | Бездротова локальна мережа | Мережа, що використовує Wi-Fi для з’єднання пристроїв, наприклад, підключення ноутбука до офісної мережі через SSID (1.1.6). |
| Bluetooth                 | Блютуз                   | Бездротова технологія для з’єднання пристроїв на коротких відстанях, наприклад, для підключення динаміка до планшета (1.1.6). |
| Network Controller        | Мережевий контролер       | Пристрій або програмне забезпечення для централізованого керування та моніторингу мережевих пристроїв через графічний інтерфейс (2.1.1). |
| Simulation Mode           | Режим симуляції           | Режим у Packet Tracer, що дозволяє зупиняти час і аналізувати мережевий трафік пакет за пакетом для перевірки підключення та функціональності (2.0.1). |
| PDU (Protocol Data Unit)  | Одиниця даних протоколу   | Пакет даних, що передається через мережу; у Packet Tracer використовується для аналізу трафіку в режимі симуляції (2.0.3). |
| DHCP (Dynamic Host Configuration Protocol) | Протокол динамічної конфігурації хоста | Протокол, що автоматично призначає IP-адреси пристроям у мережі, наприклад, комп’ютерам і принтерам в офісній мережі (1.1.4). |
| Topology                  | Топологія                 | Схема розташування та з’єднання пристроїв у мережі, яку можна створювати та редагувати в Packet Tracer (1.1.1). |

  2. **Крок 2: Підключення Office-SW1 до патч-панелі**

     Для впорядкування кабелів клацніть правою кнопкою миші на вільному місці в стійці та оберіть Manage All Cables on Rack.   5. **Крок 4: Встановлення настінного роз’єму в офісі**  
     У Device-Type Selection Box оберіть Connections > Structured Cabling > Copper Wall Mount. Клікніть на стіну поруч із Equipment Cabinet, щоб встановити роз’єм (назва повинна бути Wall Mount0). З’єднайте Wall Mount0 з Patch Panel0: Punchdown1 → Punchdown13, Punchdown2 → Punchdown14, Punchdown3 → Punchdown15, Punchdown4 → Punchdown16, використовуючи кабелі Copper Straight-Through.

На Office-Admin перейдіть до Desktop > IP Configuration > виберіть DHCP для отримання IP-адреси.      У фізичному режимі клацніть правою кнопкою на кабелі, оберіть Create BendPoint, і перетягніть чорний квадрат до стіни або підлоги, щоб кабель не перетинав кімнату. Повторіть для всіх кабелів до отримання охайного вигляду.

  8. **Крок 7: Додавання додаткового настінного роз’єму та підключення Office-User**  
     Поверніться до шафи з обладнанням і підключіть: G1/0/21 → Jack21, G1/0/22 → Jack22, G1/0/23 → Jack23, G1/0/24 → Jack24 на Patch Panel0. З’єднайте: Punchdown1 → Punchdown21, Punchdown2 → Punchdown22, Punchdown3 → Punchdown23, Punchdown4 → Punchdown24. Підключіть FastEthernet0 на Office-User до Jack1 на Wall Mount1. #### Лабораторна робота 2.1.3: Packet Tracer – Monitor Your Network Using a Network Controller

- **Мета**:  
  Реалізувати мережевий контролер для моніторингу конфігурованої мережі через графічний інтерфейс (GUI). Це включає підключення контролера, перевірку підключення та відкриття нових пристроїв для спостереження за змінами в офісній мережі.

Увімкніть контролер, якщо необхідно, натиснувши кнопку живлення. #### Лабораторна робота 2.1.5: Packet Tracer – Manage and Configure your Network using a Network Controller
- **Мета**:  
  Розгорнути мережевий контролер, виконати відкриття пристроїв та додати новий мережевий пристрій до офісної мережі, спостерігаючи за оновленнями в Dashboard контролера.

- **Хід виконання**:
